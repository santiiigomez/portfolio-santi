## 🛠️ Stack Tecnológico
- **Frontend:** React 19 + TypeScript + Tailwind CSS 4.
- **Animaciones:** Motion (framer-motion).
- **Backend:** Firebase (Firestore para base de datos en tiempo real).
- **Herramientas:** Vite, Git, Lucide Icons.

## ✨ Funcionalidades Destacadas
- **Minijuego "Debug Time":** Un desafío interactivo para eliminar bugs.
- **Leaderboard:** Tabla de puntuaciones global conectada a Firebase.
- **Formulario de Contacto:** Integración directa con Firestore.
- **Diseño Ultra-Moderno:** Estética profesional con efectos de glassmorfismo.
